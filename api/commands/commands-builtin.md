# Built-in Commands

Complete list of built-in server commands.

## Debug Commands

| Command | Description |
|---------|-------------|
| `VersionCommand` | Show version info |
| `HelpCommand` | Show help |
| `CommandsCommand` | List commands |
| `DumpCommandsCommand` | Dump command info |

## Player Commands

| Command | Description |
|---------|-------------|
| `PlayerCommand` | Player subcommands |
| `GameModeCommand` | Change game mode |
| `DamageCommand` | Damage player |
| `KillCommand` | Kill player |
| `PlayerRespawnCommand` | Respawn player |
| `PlayerResetCommand` | Reset player |
| `HideCommand` | Hide player |
| `WhereAmICommand` | Show position |
| `WhoAmICommand` | Show player info |

### Inventory

| Command | Description |
|---------|-------------|
| `GiveCommand` | Give items |
| `GiveArmorCommand` | Give armor |
| `InventoryCommand` | Inventory subcommands |
| `InventoryClearCommand` | Clear inventory |
| `InventorySeeCommand` | View inventory |
| `ItemStateCommand` | Item state |

### Player Stats

| Command | Description |
|---------|-------------|
| `PlayerStatsGetCommand` | Get stat |
| `PlayerStatsSetCommand` | Set stat |
| `PlayerStatsAddCommand` | Add to stat |
| `PlayerStatsResetCommand` | Reset stats |
| `PlayerStatsDumpCommand` | Dump all stats |

### Camera

| Command | Description |
|---------|-------------|
| `PlayerCameraSubCommand` | Camera subcommands |
| `PlayerCameraResetCommand` | Reset camera |
| `PlayerCameraTopdownCommand` | Top-down view |
| `PlayerCameraSideScrollerCommand` | Side-scroller view |

## Server Commands

| Command | Description |
|---------|-------------|
